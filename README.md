# 智能理财网页应用（Python 版）

仅需 **Python** 运行，无需 npm/node。简约科技风的个人理财：存款目标、收支记录（上传/手动）、数据分析与理财建议。

## 运行方式

1. 安装 Python（建议 3.8+）
2. 在项目目录下执行：

```bash
pip install -r requirements.txt
python app.py
```

3. **先保持该窗口不要关**，再在浏览器地址栏输入并打开：**http://127.0.0.1:8080**

## 功能说明

- **存款目标**：新建目标、查看进度、更新当前金额、删除目标
- **交易记录**：下载统一模板 → 填写 CSV → 同一入口上传；支持手动添加单条；删除记录；每页 10 条，页码仅显示前 5 页与后 3 页，中间用「上一页」「下一页」切换
- **数据分析**：选择月度/季度/年度/自定义周期；收支占比、日均金额、支出分类占比、花销建议、理财产品方向建议

## 统一上传模板

CSV 表头必须为：`交易时间,类型,金额,分类,对方/备注,来源`  
类型填「收入」或「支出」，来源填「微信」「支付宝」「招商银行」或「手动」。  
可在页面上点击「下载统一模板」获取示例 CSV。

## 数据存储

数据保存在项目目录下的 `data/` 文件夹：
- `transactions.json`、`goals.json`：交易与目标
- `fund_cache.json`：东方财富基金推荐缓存，**每月 1 日**自动更新

## 智能推荐 - 基金数据

- **数据来源**：东方财富 / 天天基金公开接口
- **更新周期**：每月 1 日自动刷新（当月首次访问时拉取）
- **推荐类型**：货币型、债券型基金，每类最多 3 只，优先光大、招商、广发、交通等
- **Token 消耗**：0（纯 HTTP 请求，不涉及 AI）
